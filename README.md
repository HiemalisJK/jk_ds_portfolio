### Economic Development and Employer Planning System (EDEPS)

| Field        | Details                          |
|--------------|----------------------------------|
| **Period**   | 2025-05-25 – 2025-07-16 (6 weeks) |
| **Tech Stack / Tools** | numpy, pandas, MySQL, Power BI, ARIMA |
| **Project Personnel**   | 3 |

Unfortunately I cannot share the entire project pipeline.\
But you can see the final [dashboard](/EDEPS/edpbs_dashboard_export.pdf) overview!

#### Objective
- Georgia Institute of Technology requested to recreate outdate EDEPS interactive dashboard
- Link Integrated Post-secondary Education Data (IPEDS) with Bureau of Labour Statistic (BLS) 
- Create re-usable data processing pipeline

#### Obstacles & Solutions
- **Obstacle:** Linkage between IPEDS and BLS data was many to many relationship.
- **Solution:** Performed extensive EDA using SQL and pandas to re-define relationships.
- **Limitation:** In process of re-definition there is potential for disagreement with classification in future.
